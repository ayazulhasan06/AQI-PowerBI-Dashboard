**Data Ingestion ‚Üí Data Cleaning ‚Üí Data Modeling ‚Üí DAX Measures ‚Üí Interactive Dashboard**

## üéØ Business Problem

Air quality data is often:
- Large and inconsistent  
- Difficult to analyze without proper categorization  
- Hard to interpret for decision-making  

This project solves these challenges by:
- Cleaning and standardizing AQI data
- Categorizing AQI based on health impact levels
- Identifying key pollutants affecting air quality
- Presenting insights through interactive dashboards

## üìò Key Features

### üîπ Data Ingestion & Transformation
- Imported day-wise AQI data from Excel
- Cleaned and transformed data using Power Query:
  - Removed null and duplicate values  
  - Standardized data types  
  - Normalized pollutant values  
  - Optimized dataset for performance  

### üîπ Data Modeling
- Built an optimized data model
- Established relationships for cross-filtering
- Enabled slicers for Year, State, and City

### üîπ AQI Categorization
AQI values are classified into:
- Good  
- Moderate  
- Poor  
- Unhealthy  
- Very Unhealthy  

### üîπ DAX Measures & KPIs
- Average AQI  
- Maximum AQI  
- Primary Pollutant  
- AQI Category  
- Pollutant occurrence frequency  

### üîπ Dashboard & Visualization
- KPI cards for key AQI metrics  
- Interactive map showing AQI by State  
- AQI trend analysis over time  
- City-wise AQI comparison  
- AQI level distribution  
- Clean, business-oriented dashboard layout
